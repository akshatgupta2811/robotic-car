# 🤖 Robotic Car

A simple Python-based robotic car project.

## Files

- `.py` – Main Python program
- `test.py` – Testing script

## Requirements

- Python 3.x

## Run

Run the main program:

```bash
python .py
```

Run the test file:

```bash
python test.py
```
